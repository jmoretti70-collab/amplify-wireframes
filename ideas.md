# Ideias de Design - Amplify Wireframes

## Abordagem Selecionada: Gradiente Vibrante com Profundidade

### Movimento de Design
Inspirado em interfaces modernas de SaaS com gradientes vibrantes (laranja para rosa), combinando elementos de glassmorphism e profundidade através de sombras suaves.

### Princípios Fundamentais
1. **Gradiente Amplify:** Uso consistente do gradiente laranja-para-rosa como identidade visual
2. **Hierarquia Clara:** Tipografia bem definida com pesos variados para guiar o olhar
3. **Espaçamento Generoso:** Whitespace estratégico para respiração visual
4. **Interatividade Sutil:** Transições suaves e feedback visual em hover states

### Filosofia de Cores
- **Primário:** Gradiente de #f97316 (laranja) para #ec4899 (rosa)
- **Background:** Cinza claro (#f8f9fa) para áreas de conteúdo
- **Cards:** Branco puro com sombras suaves
- **Texto:** Cinza escuro (#1f2937) para legibilidade

### Paradigma de Layout
- Sidebar fixa para navegação no dashboard web
- Bottom navigation fixa para o PWA mobile
- Cards com bordas arredondadas e sombras sutis
- Grid responsivo para organização de conteúdo

### Elementos de Assinatura
1. **Splash Screen Animado:** Logo pulsante com loading dots
2. **Phone Frames:** Mockups realistas de dispositivos móveis
3. **Gradiente Header:** Cabeçalho com gradiente Amplify em todas as telas

### Filosofia de Interação
- Hover states com elevação de sombra
- Transições de 200-300ms para suavidade
- Feedback visual imediato em ações

### Sistema Tipográfico
- **Display:** Font-weight 700 para títulos
- **Body:** Font-weight 400-500 para texto corrido
- **Hierarquia:** Tamanhos bem definidos (3xl, 2xl, xl, lg, base, sm, xs)
